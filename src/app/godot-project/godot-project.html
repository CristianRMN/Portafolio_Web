Videojuego hecho con Godot

Todavía en desarrollo, hecho con el motor de videojuegos Godot

DESCRIPCIÓN

Este proyecto es un videojuego en desarrollo en Godot 4.2.2,
que voy creando cuando tengo tiempo. Es un plataformas 2D
estilo Mario/Crash Bandicoot, donde se puede realizar la recolección
de cristales mágicos, incrementar la vida y la potencia de llamarada,
y enfrentarse a enemigos desafiantes. El juego cuenta con 70 niveles,
incluyendo un boss final cada 10 niveles. Se utiliza un singleton para
la persistencia de datos entre escenas, como vida, llaves y cristales,
y el guardado de datos se realiza mediante JSON. Los assets combinan
recursos gratuitos y otros generados con IA, creando un entorno visual
atractivo mientras se completa el desarrollo del juego.
