APP MUSICAL CON JAVA Y SPRING BOOT

Proyecto Musical hecho con java y Spring boot

DESCRIPCIÓN

Este proyecto es una aplicación musical desarrollada con Java y Spring Boot,
diseñada siguiendo una arquitectura de microservicios.La aplicación permite
que los usuarios se registren y hagan login para acceder a funcionalidades
completas, incluyendo la consulta de información de álbumes, artistas y canciones,
 la reproducción de música y la gestión de un historial de canciones escuchadas, así
como un top de canciones más escuchadas. La aplicación utiliza REST Controllers
para manejar los endpoints de la API, DTOs y modelos de datos para la transferencia
de información, y services para encapsular la lógica de negocio. La persistencia
de datos se gestiona mediante JPA y JPARepository, comunicándose con PostgreSQL
para mantener las relaciones entre tablas, mientras que MongoDB se utiliza para almacenar datos
que no requieren relaciones, como el historial y el top de canciones. El proyecto está estructurado
en tres microservicios: uno encargado de la entrada de datos, otro para la comunicación
con la base de datos relacional y un tercero para la gestión de los datos en MongoDB
, asegurando escalabilidad, modularidad y un manejo eficiente de la información.
