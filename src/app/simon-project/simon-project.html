Simon dice en Kotlin

App para móvil que simula el juego de simon dice. Hecho con Android Studio

DESCRIPCIÓN

Este proyecto es una versión del clásico juego “Simón Dice”, desarrollada en Android Studio
 utilizando Kotlin. La aplicación recrea la dinámica del popular juego de memoria, donde
el usuario debe repetir correctamente una secuencia de colores que se ilumina de forma progresiva.
Al iniciar la aplicación, el usuario puede pulsar el botón “Start” para comenzar la partida.
El juego muestra primero un color aleatorio, y si el jugador acierta, la secuencia aumenta
en longitud, repitiendo los colores anteriores y añadiendo uno nuevo. Por ejemplo,
si primero se ilumina el rojo, en la siguiente ronda se iluminarán el rojo y el naranja,
y así sucesivamente. El juego incluye un conteo de rondas, aciertos y récords,
que se actualizan dinámicamente a medida que el jugador progresa. La interfaz
de usuario está desarrollada con Jetpack Compose, ofreciendo una experiencia moderna
y fluida. La iluminación secuencial de los colores se gestiona mediante corutinas,
y la UI se actualiza en tiempo real utilizando MutableState, garantizando una interacción
reactiva y sin interrupciones.
